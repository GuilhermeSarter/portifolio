Minha Biblioteca: Uma Webpage Personalizada

Este projeto é uma página web desenvolvida com HTML e CSS, com o objetivo de apresentar um super meio-campista que está livre no mercado e à procura de um novo time. A página exibe informações detalhadas sobre o jogador, incluindo suas habilidades, histórico de clubes, estatísticas e conquistas, além de contatos para negociações.

A estrutura do site é construída em HTML para organizar o conteúdo de forma semântica e clara, enquanto o CSS é utilizado para dar estilo e tornar a apresentação visualmente atraente e profissional, destacando os pontos fortes do atleta.

O layout é dividido em seções, começando por um cabeçalho com o nome do jogador e seu slogan, seguido de uma área principal (.container) que contém um perfil detalhado, imagens em alta resolução e uma seção com vídeos dos melhores momentos. A navegação é facilitada por um menu fixo e o design responsivo garante que a página se adapte a diferentes dispositivos e tamanhos de tela.

O CSS aplica cores sóbrias e elegantes, utilizando variáveis para manter a consistência visual, fontes modernas importadas do Google Fonts, e efeitos de hover para tornar a experiência do usuário mais interativa. A simplicidade no uso de HTML e CSS permite que a página seja rápida, acessível e focada na experiência do visitante, ajudando a destacar o talento do meio-campista para possíveis clubes interessados.
